Bem-vindo ao NewMoney

Um sistema financeiro simples para cadastrar ganhos e gastos, acompanhar o saldo e navegar com facilidade em qualquer dispositivo.

Recursos

  • Cadastro de ganhos e gastos
  • Painel de dashboard financeiro
  • Edição e exclusão de transações
  • UI responsiva para desktop e mobile

Como começar

  • Use o menu para navegar entre as páginas
  • Adicione transações em Ganhos ou Gastos
  • Acompanhe seus resultados no Dashboard
