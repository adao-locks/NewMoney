Sobre o Projeto

Este projeto foi criado como um sistema financeiro básico para controle de ganhos e gastos, com foco em usabilidade e responsividade.

O que está incluso

  • Roteamento entre páginas com Angular Router
  • Componentes standalone
  • Formulários com feedback visual
  • Persistência via localStorage

Benefícios

  • UI limpa e moderna
  • Fácil navegação
  • Compatível com dispositivos móveis
  • Estrutura modular para expansão
