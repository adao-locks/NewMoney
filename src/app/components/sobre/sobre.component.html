Sobre o Projeto

Este é um projeto de exemplo que demonstra como criar uma aplicação Angular moderna com sistema de rotas.

Tecnologias Utilizadas:

  • Angular 19+
  • TypeScript 5.6+
  • CSS3
  • RxJS 7.8+

Características:

✨ Componentes Standalone - Não requer NgModules

🚀 Roteamento Avançado - Sistema de rotas completo

📱 Design Responsivo - Funciona em todos os dispositivos

⚡ Performance Otimizada - Aplicação leve e rápida
